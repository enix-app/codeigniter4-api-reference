getVarData()

Gets a collection of data that should be shown in the 'Vars' tab.

The format is an array of sections, each with their own array of key/value pairs:

$data = [

 'section 1' => [
     'foo' => 'bar,
     'bar' => 'baz'
 ],
 'section 2' => [
     'foo' => 'bar,
     'bar' => 'baz'
 ],

];
